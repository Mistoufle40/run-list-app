# run-list-app

## Brief : 

This project is a simple project to see a list of race with details and visualization of differents steps of the race.

## How to use : 

In order to launch the project, go to the run-list-app directory and run *```npm install```*  and *```npm start```*.

Create a new .env file at the root of the directory run-list-app and create a new environement variable named : *```REACT_APP_GOOGLE_API_KEY```* and give it your Google Maps API key (example: REACT_APP_GOOGLE_API_KEY="YOUR_API_KEY").

This file allow you to see the rendering of the points on Google Maps component. 

If you don't have Google API key, you can follow this documentation : https://developers.google.com/maps/documentation/javascript/get-api-key.
